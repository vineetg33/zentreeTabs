Grouping Threshold

The Grouping Threshold controls how many tabs from the same domain must be open before ZenTree's AI automatically groups them together. This setting helps keep your workspace organized by preventing single tabs from being grouped unnecessarily.

How It Works

When you use the AI Organize feature, ZenTree analyzes your open tabs and looks for patterns:

  • Domain Grouping: Adjacent tabs from the same website (e.g., multiple GitHub pages)
  • Semantic Grouping: Tabs with related content based on AI analysis
  • Smart Hybrid: Combines both methods for optimal organization

Threshold Options

2+ tabs (Aggressive) If you have 2 or more tabs from the same domain (e.g., two YouTube videos), they'll be grouped together. Best for users who want maximum organization.
3+ tabs (Balanced - Default) Requires at least 3 tabs from the same domain before grouping. Best for most users - balances organization without over-grouping.
4+ tabs (Conservative) Only groups domains with 4 or more tabs. Best for users who prefer minimal grouping.
5+ tabs (Minimal) Only groups domains with 5 or more tabs. Best for users working with many different sites simultaneously.

When to Adjust This Setting

  • Lower threshold (2-3): If you often research topics with multiple tabs from the same site
  • Higher threshold (4-5): If you prefer manual organization or work across many different websites

AI Features

ZenTree uses local AI models to intelligently organize your tabs without sending any data to external servers. All processing happens directly in your browser for privacy and speed.

Enable AI Organize

Toggle this setting to enable or disable the AI-powered tab organization feature. When enabled, you can click the sparkle (‚ú®) button in the footer to automatically group your tabs.

Smart Hybrid Mode

ZenTree's AI uses a hybrid approach that combines:

  • Domain-first clustering (controlled by the Grouping Threshold)
  • Semantic similarity analysis using natural language processing
  • Proximity weighting (tabs opened close together are more likely to be related)
  • Temporal coherence (recently opened tabs get higher relevance scores)
