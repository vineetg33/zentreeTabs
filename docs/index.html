ZenTree Tabs

Organize your browser tabs efficiently with tree-based hierarchy and AI-powered grouping.

Install Extension View on GitHub

Features

Tree-Based Organization

Tabs automatically organize into a hierarchical tree structure based on how you opened them. See parent-child relationships at a glance.

AI-Powered Grouping

Smart hybrid AI analyzes tab content and semantically groups related tabs. One-click undo if you don't like the results.

Quick Search

Instantly find any tab by title or URL. No more hunting through dozens of tabs to find what you need.

Keyboard Shortcuts

Multi-select with Ctrl/Cmd+Click, range select with Shift+Click, and middle-click to close. Built for power users.
