# Privacy Policy for ZenTree Tabs

**Last updated: January 17, 2026**

## Data Collection
ZenTree Tabs does not collect, store, or transmit any personal data outside of your local browser.

## Local Storage
The extension stores user preferences (such as UI settings and tree state) locally in your browser using Chrome's storage API. This data never leaves your device.

## Permissions
- **tabs, tabGroups**: Read tab information to build the tree hierarchy
- **sidePanel**: Display the tab tree interface
- **storage**: Save your preferences locally
- **favicon**: Display website icons
- **bookmarks**: Manage bookmarks from the extension
- **downloads**: Export tab session data

## Third Parties
ZenTree Tabs does not share any data with third parties.

## Contact
For questions about this privacy policy, please open an issue at: https://github.com/shuknuk/zentreetabs/issues
